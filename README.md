# Посмотри в окно

Вторая проектная работа на курсе Фронтенд разработчик. Страница с реализацией поиска по видео по времени суток и городу

## Функциональность

- Поиск по городу и времени суток
- Список карточек с городами
- Кастомные элементы формы (чекбоксы, поле ввода)
- Состояния элементов: active, hover, focus,focus-visible

## Технологии

- HTML5
- CSS3
- JavaScript
